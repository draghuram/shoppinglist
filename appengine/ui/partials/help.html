Help



  • All the functionality of this web application revolves around the concept of a shopping Item.
  • The only requirement for an item is its name. It can optionally belong to a single category. It can also have a list of stores that are usually the places where it is purchased.
  • You need to first add items to the database before carrying out any other operation. This is done in by using New item link in the navigation bar. Once an item is created, it can be added to the shopping list or removed from such list in Items page.
  • Items added to the shopping list show up in List page. This is the page you would use when you are actually in a store. Items can be removed from the list either individually or as a group (by selecting such items and clicking on the button Clear selected items).
  • For source code and other details about the app, please visit "github" link.
